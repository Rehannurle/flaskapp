About FarmsSphere

Student Project

FarmsSphere is a final year project developed by BSc IT students with a vision to revolutionize agricultural practices through technology. Our goal is to provide farmers with data-driven insights and recommendations to optimize their farming operations.

Important Notice

Please Note: This application is a student project and should be used for educational and demonstration purposes only. While we strive for accuracy, the predictions and recommendations should not be used as the sole basis for agricultural decisions.

Data Privacy: Please do not input any sensitive or personal information into this application. Use sample or non-sensitive data for testing purposes.

Project Features

Our application offers several key features designed to assist farmers:

  • Rainfall Prediction using historical weather data
  • Crop Recommendation based on soil and climate conditions
  • Yield Prediction to help with harvest planning

Technology Stack

This project is built using modern technologies and machine learning algorithms to provide accurate predictions and recommendations. We've utilized Python, Flask, and various data science libraries to create a robust agricultural assistant.
